ALUGU CONSULTING WEBSITE - PROJECT STRUCTURE
============================================

Complete Next.js 14+ consulting website with TypeScript and Tailwind CSS.

FOLDER STRUCTURE:
-----------------

Alugu Website/
├── app/
│   ├── layout.tsx              # Root layout with Nav, Footer, metadata
│   ├── page.tsx                # Home page (all sections in order)
│   ├── globals.css             # Design tokens & global styles
│   ├── robots.ts               # SEO: robots.txt generator
│   ├── sitemap.ts              # SEO: sitemap.xml generator
│   ├── about/
│   │   └── page.tsx            # About page
│   ├── services/
│   │   └── page.tsx            # Services page
│   ├── contact/
│   │   └── page.tsx            # Contact page
│   └── api/
│       └── contact/
│           └── route.ts        # Contact form API endpoint
│
├── components/
│   ├── Nav.tsx                 # Navigation component
│   ├── Footer.tsx               # Footer component
│   ├── CTAButton.tsx            # CTA button with routing logic
│   ├── Section.tsx              # Section wrapper component
│   ├── ContactForm.tsx          # Contact form with validation
│   └── ScrollToServicesButton.tsx  # Client scroll button
│
├── lib/
│   ├── constants.ts             # App constants (CTAs, nav links)
│   └── content.ts               # Content objects for all pages
│
├── public/
│   └── favicon.ico              # Placeholder favicon
│
├── package.json                 # Dependencies & scripts
├── tsconfig.json                # TypeScript configuration
├── next.config.js               # Next.js configuration
├── tailwind.config.js           # Tailwind CSS configuration
├── postcss.config.js            # PostCSS configuration
├── .gitignore                   # Git ignore rules
├── env.example                  # Environment variables example
└── README.md                    # Complete setup & deployment guide

--------------------------------

1. Hero (H1, subheadline, primary + secondary CTAs, trust line)
2. Value Proposition (3 Pillars)
3. Problems Solved (4-6 pain points)
4. Proof Lite (3 metric cards)
5. Framework / Method (5 steps)
6. Mini About (2-4 sentences + link)
7. Final CTA Band (primary + secondary CTAs)

ABOUT (app/about/page.tsx):
1. About Hero
2. Story / Background
3. Operating Principles (5 principles)
4. Credibility / Experience (bulleted highlights)
5. CTA Section (primary CTA)

SERVICES (app/services/page.tsx):
1. Services Hero
2. Service Offerings (3 services with cards)
3. Engagement Workflow (6 steps)

CONTACT (app/contact/page.tsx):
